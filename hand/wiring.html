ロボットハンド配線図

Arduino Uno ＋ Hiwonder LFD-01 サーボ ×5（5本指独立制御）／外部6V電源

オレンジ＝信号(PWM) → Dピン 赤＝＋6V 茶/黒＝GND ＝接続点（交差で●が無い所は繋がっていない）
★ 共通GNDを必ず繋ぐ： Arduino の GND と 外部電源の − を結線する。 これが無いと信号の基準がズレてサーボが正しく動かない（一番ハマるポイント）。
サーボ電源は外部6Vのみ： 赤線を Arduino の 5V から取らない。 5本同時で最大3.5A流れ、Arduinoのレギュレータが焼ける/落ちる。
給電の順番： ①サーボ3線をArduino/レールへ配線 → ②ArduinoをUSBでPCへ（ロジック電源）→ ③外部6Vを投入 → ④スケッチ書込 → ⑤シリアルモニタ(9600/Newline)で 0 2000。
