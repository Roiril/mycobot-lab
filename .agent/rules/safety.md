# 安全規約（必読）

myCobot 320-M5 は 6-DoF 卓上アームで、24V 120W、リーチ約 350mm。意図しない動作で人や物を傷つける可能性がある。**新しいモーションを書く前に必ずこのファイルを読む**。

## 動作前の必須チェック

1. **緊急停止ボタン**が手の届く位置にあり、解除されている。
2. アームの可動範囲内に**指・手・顔・割れ物**がない。
3. アーム土台が机に**確実に固定**されている（外部電源コードに引っ張られない）。
4. ケーブル類（電源・USB）が関節に挟まらない取り回し。
5. 既知の姿勢から遠い角度にいる時は、まず HOME へゆっくり戻してから動かす。

## 速度ポリシー

| 状況 | 速度 |
|---|---|
| 通常動作 | **30 以下** |
| デモ・大きな振り | 35–50 |
| 緊急回避・低速確認 | 5–15 |
| **使用禁止域** | **80 超**（`Arm.MAX_SPEED` でガード）|

`Arm.move()` の `speed` 引数を省略すると `DEFAULT_SPEED=30`。

## 角度の安全範囲

物理的に届く範囲は機種仕様（J1±170°, J2±120°, J3±150°, J4±145°, J5±165°, J6±175°）だが、**実環境では机面・自分の体・ケーブル**との干渉が支配的。新しい姿勢は **HOME から +20° 刻みで段階的に確認**し、`poses.py` に名前で登録してから本採用すること。

## 終了処理

- スクリプト終了時は**必ず REST 姿勢（折り畳み）または HOME に戻してから**切断する。
- `release_all_servos()` を呼ぶと**アームは脱力して落下**する。机面まで届く姿勢で実行するか、人が支えること。
- 電源を切る前にも安全姿勢へ。

## 異常時

- 異音・過熱・想定外の動き → 即 **STOP ボタン**を押す。
- 通信不通 → スクリプトを `Ctrl+C` で停止。アームはその場で停止保持される。
- M5 画面が `Atom: no` → STOP 押下確認 → 電源 OFF/ON → Connect test 再実行。

## やってはいけないこと

- `send_angles` の引数を確認せずに実行する。
- 速度 > MAX_SPEED を一時的にでも使う。
- 緊急停止に手が届かない位置（部屋を離れる等）で動作させる。
- 子供やペットがいる空間で電源 ON 放置する。
